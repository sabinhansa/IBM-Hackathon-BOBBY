# BobFlow Sample Reports

This directory contains example reports generated by IBM Bob for demonstration purposes.

## Files

- **onboarding-sample.md** - Repository onboarding guide (Ask Mode)
- **architecture-sample.md** - System architecture documentation (Plan Mode)
- **doc-gaps-sample.json** - Documentation gap analysis (Ask Mode + Review Mode)
- **test-plan-sample.md** - Comprehensive test plan (Code Mode)
- **risk-review-sample.json** - Security and code quality risk assessment (Review Mode)
- **pr-pack-sample.md** - Complete PR documentation package (Code Mode + Plan Mode)

## Usage

These sample files demonstrate the quality and format of outputs that BobFlow workflows can generate. They are based on a fictional e-commerce platform but represent realistic, production-quality documentation.

## Note

All sample data is fictional and created for demonstration purposes only. No real company code or sensitive information is included.
