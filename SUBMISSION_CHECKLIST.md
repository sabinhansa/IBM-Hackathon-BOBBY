4. Place in `/bob_sessions/` directory
5. Commit and push to repository

## 🔒 Security & Safety Check

### Critical: No Secrets in Repository

- [ ] **Search for API keys**: `git grep -i "api.key\|apikey\|api_key"`
- [ ] **Search for tokens**: `git grep -i "token\|secret\|password"`
- [ ] **Check .env files**: Ensure `.env` is in `.gitignore`
- [ ] **Review .bobignore**: Confirms sensitive files excluded
- [ ] **Check commit history**: No secrets in past commits
- [ ] **Verify environment variables**: Only in documentation, not hardcoded

### Files to Double-Check

- [ ] `.env` - Should NOT exist in repo
- [ ] `.env.example` - Should only have placeholders
- [ ] `runner/examples/env-template.txt` - Only template, no real values
- [ ] All `.sh` scripts - No hardcoded credentials
- [ ] `bobflow-runner.js` - No API keys

### If Secrets Found

1. **DO NOT** just delete and commit
2. Use `git filter-branch` or BFG Repo-Cleaner to remove from history
3. Rotate any exposed credentials immediately
4. Document the incident and remediation
